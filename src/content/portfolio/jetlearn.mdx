## <a name="anchor3" />Процесс

<a href="/images/portfolio/jetlearn/1.png" target="_blank">![Платформа для корпоративного обучения | SobakaPav.ru](~/assets/images/portfolio/jetlearn/1.png)</a>
<emp>Информационная карта</emp>

Мудборд. 
Референсы-антонимы
Чтобы заказчику было проще описать пожелания к стилю, распределили референсы по восьми доскам. На каждой показали полярно разные примеры
8 оппозиций:
- Визуальный \ Текстовый
- Деловой \ Непринужденный
- Темный \ Светлый
- Красочный \ Монохромный
- Плотный \Воздушный
- Большие заливки \ Точеечные акценны
- Плоский \ С тенями
- Скругленный \ Прямоугольный
