## <a name="anchor2" />Дизайн-задачи

1. Создать концептуальные макеты, описывающие продуктовые гипотезы, подготовить макеты для проведения customer development на реальных пользователях.

2. Совместно с заказчиком разработать интерфейс для новых продуктовых фич.

3. Вписать новую функциональность с существующее решение.

4. Провести редизайн МИС «Авиценна» для веб-отображения с учетом новых продуктовых и визуальных наработок.
