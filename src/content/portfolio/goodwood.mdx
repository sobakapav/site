Исполнитель обязуется выполнить по заданию Заказчика следующие услуги: юзабилити аудит
текущего решения продукта Заказчика (сайт https://www.gwd.ru), в частности:
1.1.1. Провести обследование web интерфейса сайта (в нескольких популярных разрешениях экрана).
1.1.1.1. Провести оценку восприятия сайта пользователями
1.1.1.2. Зафиксировать паттерны при работе пользователя с сайтом
1.1.2. Составить отчёт, в который войдут следующие параметры и рекомендации.
1.1.2.1. Состояние системы на момент проведения аудита.
1.1.2.1.1. Главной страницы сайта.
1.1.2.1.2. Информационной схемы сервиса и структуры навигации.
1.1.2.2. Список общих рекомендаций по улучшению пользовательских характеристик (usability)
интерфейсов сайта.
1.1.2.2.1. Страницы, которые нужно перепроектировать.
1.1.2.2.2. Тексты, которые нужно переписать.
1.1.2.2.3. Навигационные элементы, которые следует изменить.
1.1.2.2.4. Список работ, которые стоит провести.
1.1.2.2.5. Сущности, которые желательно создать.
