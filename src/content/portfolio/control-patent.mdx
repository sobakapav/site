## <a name="anchor1" />Бизнес-задача

Государство собирает с безвизовых трудовых мигрантов ежемесячную плату за патент — документ, который дает право работать на территории России. И государство старается максимально упростить процедуру оплаты патента.

Существует проблема: не каждый мигрант знает русский язык.

Новое приложение поможет пользователям отчитываться о выполнении предписаний по устранению недостатков без визитов инспекций и инспекторов.

## <a name="anchor2" />Дизайн-задача

Компания Data Solution обратилась к нам, чтобы мы разработали дизайн-решение для Android-смартфонов, которое поможет мигрантам оплачивать патенты через банкомат, приложение банка или с помощью сотрудника отделения — даже без знания русского языка.

## <a name="anchor3" />Результат

Приложение-инструкция с генератором QR-кодов, которое поможет человеку, не знающему русского языка, оплатить патент. Пользователю достаточно в приложении набрать цифрами сумму оплаты и приложение создаст QR-код для оплаты. Затем пользователю нужно показать экран своего смартфона с QR-кодом сотруднику банка, в камеру банкомата или в камеру второго устройства, где есть банковское приложение и после осуществления оплаты сфотографировать чек и загрузить его в приложение.

### <a name="anchor4" />Интерфейсные решения

В приложении для мигрантов есть карта с расположением отделений банков и банкоматов.

### <a name="anchor5" />Простая оплата
Достаточно показать в камеру QR-код, и оплата пройдет автоматически. Внутри кода уже зашифрована сумма платежа. Если ее изменить, QR-код тоже поменяется.

### <a name="anchor6" />Подтверждение оплаты
Государственные системы пока не успели связать в единый механизм базу данных мигрантов, систему генерирования QR-кодов в приложении и онлайн-платежи. Поэтому пользователь должен прикреплять банковские чеки в приложении, чтобы подтверждать платежи. Вынужденная кибербюрократия.

![Приложение для работы с трудовыми пантентами. Загрузить фото чека | SobakaPav.ru](~/assets/images/portfolio/control-patent/4.png)

Мы предельно упростили процедуру, но на «другом конце» всё равно кто-то должен проверять чеки: модератор или нейросеть, чтобы избежать риска загрузки одного и того же чека разными людьми. За помощью в разработке подобной нейросети мы посоветовали «Контролю» обратиться к российским разработчикам ABBYY Finereader и Miro — у обеих компаний есть технологии распознавания текста.

![Приложение для работы с трудовыми пантентами. История платежей | SobakaPav.ru](~/assets/images/portfolio/control-patent/5.png)

### <a name="anchor7" />Авторизация по телефону и синхронизация с базой данных мигрантов
В приложении нет регистрации. Пользователи регистрируются сразу в органах миграции — там им выдают пароль, привязанный к номеру телефона. Их и нужно ввести при авторизации.

![Приложение для работы с трудовыми пантентами. Экран смены пароля | SobakaPav.ru](~/assets/images/portfolio/control-patent/6.png)

Заполнять данные о себе в приложении не надо — все подтянется само из  баз данных органов миграционного учёта. Отредактировать такие данные нельзя, но можно загрузить фотографию.
