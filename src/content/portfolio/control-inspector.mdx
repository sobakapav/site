## <a name="anchor3" />Результат

Сервис для удаленного проведения проверок контрольными органами подотчетных им компаний. 

Дедали три продукта: десктопный веб для инспектора, мобильное приложение для инспектора, мобильное приложение для контролируемого лица.

Аналитика была на стороне заказчика, мы работали по строгому ТЗ и привлекали команду заказчика только в случае непонятных нам отраслевых и юридических моментов.

Важно было сохранить преемственность государственных дизайн-систем и сделать минимум кастомного для быстрой разработки.

Важными для работы с сервисом было удобно организовать напоминание о предстоящих встречах (и для инспектора, и для КЛ), чтобы свести к минимуму опоздания и дискоммуникацию.

Крутой кусок системы – видеозвонки (ВКС), во время которых происходят проверки. В процессе встречи инспектору важно следить за местоположением представителя контролируемого лица, видеть список объектов для контроля и взаимодействовать со списком вопросов из чек-листа проверки.

Еще любопытный функционал – возможность передать КЛ на время чек-лист для самостоятельного заполнения (в случае, если на объекте нет связи и ВКС невозможна). Тогда процесс ответов на вопросы чек-листа становится доступен КЛ. В этом режиме предусмотрели кейс отложенной загрузки данных (пока на устройстве не появится связь, данные хранятся локально и пытатся отправиться на сервер).

По итогу проекта часть предложенного функционала пришлось перенести в бэклог, а первую версию для показа в министерстве собирать из самых важных (по мнению министра) элементов и функциональности.
