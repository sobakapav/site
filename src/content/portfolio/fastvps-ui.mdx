## <a name="anchor3" />Результат

Пересобрали шаблоны настроек. Сделали несколько решений — и для тех, кому сложно разбраться в процессе, и для тех, кто чувствует себя во всем этом как рыба в воде.

Для первых — преднастройки, чтоб можно было не глядя выбрать, то, что предлагает FastVPS. Для вторых — пошаговые настройки с пояснениями.
