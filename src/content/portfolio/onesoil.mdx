## <a name="anchor3" />Процесс

Проверили, насколько выполняются ключевые задачи пользователей в приложении OneSoil. Для этого сформулировали цели и ключевые задачи пользователя.

В рамках первой цели — «Разобраться в приложении» — у пользователя есть две задачи:
- Ознакомиться с возможностями приложения
- Добавить первое поле

Вторая цель — «Следить за состоянием своих полей» — вызывает следующие задачи:
- Добавить свои поля
- Добавить информацию о своих полях
- Следить за состоянием поля
- Фиксировать заметки
- Принимать решения

Дополнительные задачи:
- Авторизоваться
- Обратиться за помощью

_Детализация задач_

Нашли и описали более шестидесяти интерфейсных проблем, ранжировали их по важности. Предложили для каждой одно или несколько решений.

<a href="/images/portfolio/onesoil/1.png" target="_blank">![Мобильное приложение для фермеров | SobakaPav.ru](~/assets/images/portfolio/onesoil/1.png)</a>
_Одна из более чем шестидесяти интерфейсных проблем на доске в Miro_

Четыре сквозные проблемы: 
- Переключатель видов
- Выделение полей, добавление.
- Поиск как дополнительный вектор
- Неточности в формулировках
