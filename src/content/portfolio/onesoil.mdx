<a href="/images/portfolio/onesoil/54.png" target="_blank">![UX-аудит мобильного приложения для фермеров • Собака Павлова](~/assets/images/portfolio/onesoil/54.png)</a>

## <a name="anchor4" />Особенности процесса

Сформулировали цели и ключевые задачи пользователя, чтобы проверить, насколько они выполняются в приложении OneSoil.

<a href="/images/portfolio/onesoil/2.png" target="_blank">![UX-аудит мобильного приложения для фермеров • Собака Павлова](~/assets/images/portfolio/onesoil/2.png)</a>
_Детализация задач_

В рамках первой цели — «Разобраться в приложении» — у пользователя есть две задачи:
- Ознакомиться с возможностями приложения
- Добавить первое поле
