## <a name="anchor1" />Бизнес-задача

Произвести оценку существующего интерфейса, подготовить рекомендации для редизайна, выполнить редизайн существующего интерфейса в критически важных местах.

Заказчик по результатам редизайна ожидает увеличить количество совершенных сделок.

## <a name="anchor2" />Дизайн-задача

- Провести оценку существующего интерфейса и свериться с менеджерами на интервью, обсудить с ними проблемы и сложности, возникающие при работе с приложением.
- Подготовить список рекомендаций по улучшению интерфейса, и на его основе создать презентацию для руководства заказчика.
- Выполнить редизайн существующего интерфейса в критически важных местах, опираясь на список выявленных проблем.
- Провести тестирование обновленного интерфейса приложения, зафиксировать результаты.д

## <a name="anchor3" />Результат

62 комментария по результатам аудита существующего интерфейса. Построили карту пути пользователя в интерфейсе, вынесли на карту комментарии с описанием проблемы и добавили варианты решения ко всем выявленным проблемам
7 пользовательских интервью. Провели интервью не только с непосредственными пользователями приложения (менеджеры), но и с сотрудниками “офиса” (специалистами ведения сделок, согласующими менеджерами). Это позволило посмотреть на проблемы с разных сторон и помогло подсветить проблемы, которые считались только интерфейсными.
5 групп замечаний и проблем. На основе всей полученной информации с аудита и интервью, команда Собаки провела воркшоп по выработке рекомендаций по улучшению интерфейса. Проблемы объединили в группы замечаний. Такое разбиение на группы в дальнейшем поможет заказчику устранять проблемы не только со стороны интерфейса.
