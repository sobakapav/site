## <a name="anchor1" />Бизнес-задача

Создать удобную современную версию медицинской информационной системы на основе существующей: измениить визуальное решение, улучшить UX и привести интерфейс в соответствие с реальным уровнем развития продукта.

## <a name="anchor2" />Дизайн-задача

Провести редизайн МИС, переработать процесс первичного приема пациента. Разработать визуальный стиль, нарисовать новую навигацию, продумать UX-решение. Создать детальные макеты интерфейса в дизайне и собрать библиотеку интерфейсных компонентов (UI kit), при помощи которой компания «Онкор» может собирать остальные макеты самостоятельно.

## <a name="anchor3" />Результат

Разработали концептуальное навигационное решение, модернизировали карту первичного осмотра пациента, подготовили компоненты для реализации остальной функциональности.
