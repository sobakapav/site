- Провели аудит интерфейса
- Переработали веб-версию.
- Спроектировали мобильное приложение.

Delivered to customer:

Insightful UX expert evaluations
Sprint-based design synced with the dev team
Design system based on capabilities of the framework

We embraced an agile approach, syncing with the dev sprints. Our tasks were widely different: from switching to a different framework and making a mobile version to enhancing functionality and streamlining user journeys. We had to be as flexible as a regular startup usually is.
