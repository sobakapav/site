## <a name="anchor1" />Бизнес-задача

Автоматизировать процессы управления IT-инфраструктурой через улучшение пользовательского взаимодействия с системой.

<em>Дашборд. Вариант 1</em>

## <a name="anchor2" />Дизайн-задача

Создать понятный и функциональный интерфейс для системы управления IT-инфраструктурой:
1. **Проанализировать текущий интерфейс** — выявить основные проблемы взаимодействия пользователей с системой, включая сложности навигации, перегруженность интерфейса и недостаточную информативность экранов.
2. **Разработать новую информационную архитектуру** — оптимизировать структуру разделов, перераспределить функциональные блоки и создать логичную навигацию.
3. **Адаптировать интерфейс под Ant Design** — использовать существующие компоненты библиотеки для унификации элементов, ускорения разработки и соблюдения визуального единства.
4. **Создать UX-решения для разных типов пользователей** — учесть специфику работы администраторов и других ролей в системе, обеспечив удобный доступ к нужным инструментам.
5. **Разработать интуитивно понятные интерфейсы** — минимизировать когнитивную нагрузку, улучшить читаемость данных и упростить взаимодействие с системой для ускорения рабочих процессов.
